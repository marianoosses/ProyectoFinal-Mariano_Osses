# Proyecto Final

Página Web destinada a usuarios que quieran compartir opiniones sobre películas y series.

El usuario podrá realizar las siguientes acciones:

- Login
- Crear Usuario
- Cerrar sesión
- Agregar películas
- Agregar series
- Calificar películas
- Calificar series
